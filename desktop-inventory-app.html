Sobre el Proyecto

Este proyecto es una aplicación de escritorio (Windows Forms) para la gestión de artículos, desarrollada en C# como práctica completa de un sistema tipo ABM (Alta, Baja y Modificación). El objetivo fue construir una app realista de administración de productos, trabajando la conexión a base de datos, la interacción con grillas, validaciones y un flujo de uso claro para el usuario.

La pantalla principal muestra un listado en tabla con los artículos cargados (código, nombre, descripción, marca, categoría y precio). Desde esa vista se pueden ejecutar acciones rápidas para agregar, modificar, eliminar y ver detalles de cada artículo, manteniendo una experiencia simple y ordenada. Además, al seleccionar un registro, la aplicación muestra la imagen asociada para facilitar la identificación visual del producto.

Para mejorar la búsqueda dentro del catálogo, se implementó un filtro rápido (texto libre) y un filtro avanzado que permite elegir el campo por el cual buscar, el operador (por ejemplo “contiene”, “comienza con”, etc.) y el valor a filtrar. Esto permite encontrar artículos de forma más eficiente incluso cuando el listado crece.

  • Listado completo: grilla con información clave del artículo (código, marca, categoría, precio, etc.).
  • ABM de artículos: alta, modificación y eliminación de productos desde botones dedicados.
  • Vista de detalles: pantalla para visualizar información del artículo seleccionado sin editarlo.
  • Filtros: filtro rápido por texto y filtro avanzado por campo + operador + valor.
  • Imágenes: previsualización de imagen del artículo al seleccionar un registro.
  • Validaciones: control de datos antes de guardar cambios para evitar inconsistencias.
  • Persistencia: conexión a base de datos para guardar y consultar artículos, marcas y categorías.
  • Arquitectura: separación de responsabilidades (dominio / acceso a datos / interfaz) para mantener el código más ordenado.
