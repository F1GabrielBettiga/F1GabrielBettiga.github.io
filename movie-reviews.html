Plataforma de Reseñas de Películas

Aplicación web de reseñas de películas con registro e inicio de sesión de usuarios, creación de opiniones y gestión de contenido. Desarrollada con ASP.NET MVC, Entity Framework e Identity, incorporando además asistencia mediante IA para la generación de reseñas.

Sobre el Proyecto

Esta aplicación es una plataforma web de reseñas de películas creada con el objetivo de practicar y aplicar un flujo completo de desarrollo web: desde la persistencia de datos y la arquitectura del proyecto hasta la interfaz de usuario y la experiencia de uso. El sistema permite a los usuarios registrarse, iniciar sesión y participar dejando reseñas y puntuaciones sobre películas disponibles en la cartelera.

La aplicación está organizada en módulos y pantallas para separar claramente las responsabilidades. Incluye navegación por secciones, listados con filtros/búsquedas, vistas de detalle y formularios de alta/edición. Se implementó autenticación y autorización con ASP.NET Identity para controlar el acceso según el rol del usuario, diferenciando funcionalidades de usuario común y administrador.

A nivel de funcionalidad, el usuario puede explorar el catálogo de películas, ingresar a la vista de detalle y ver reseñas existentes. Además puede crear su propia reseña asignando una calificación y un comentario, y administrar su información de perfil. Por otro lado, el rol administrador cuenta con pantallas para gestionar el contenido: alta, baja y modificación de películas, categorías y usuarios.

  • Autenticación completa: registro, login, logout y manejo de sesión con ASP.NET Identity.
  • Roles y permisos: separación de accesos entre usuario y administrador.
  • Películas: listado de cartelera, vista de detalle y CRUD administrativo.
  • Reseñas: creación de reseñas con puntuación y comentario, visualización en el detalle de la película.
  • Categorías: administración de categorías y asociación con películas.
  • Usuarios: gestión de usuarios desde panel administrador y edición de perfil.
  • Persistencia: Entity Framework + SQL Server, con relaciones entre entidades y validaciones.
  • Interacción dinámica: uso de Fetch API / AJAX para acciones sin recargar la página (por ejemplo en la creación de reseñas mediante modal y vista parcial).
